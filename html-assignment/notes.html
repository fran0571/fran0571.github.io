List of 5 HTML tags

The < img > tag
It's used to display images on the web page. Its alt attribute is useful for accessibility: < img src="images/photo.jpg" alt="Dogs and Cats image" >
The < a > anchor tag
It's used to create a hyperlink on the webpage to link one page to another. Its href attribute indicates the link's destination: < a href="index.html" > Home < /a >
The < div > tag
It's used to define a division or a section. The content is written between the opening < div > and closing < /div > tags.
The < nav > tag
It's used to represent a section which contains navigation links.The navigation content is written between the opening < nav > and closing < /nav > tags.
The < p > tag
It's used to define a paragraph of text on the web page. The text followed by the tag creates a paragraph on the page.
